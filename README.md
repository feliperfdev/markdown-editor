# Markdown Editor

A Markdown flexible editor designed for Flutter.

![example_1](assets/screenshots/example_1.png)

#### Markdown options

| Markdown          | Available |     |     |     |
| ----------------- | --------- | --- | --- | --- |
| # Header          | ❌        |     |     |     |
| **Bold**          | ✅        |     |     |     |
| _Italic_          | ✅        |     |     |     |
| ~~Strikethrough~~ | ✅        |     |     |     |
| Underline         | ✅        |     |     |     |
| • Bullet list     | ✅        |     |     |     |
| 1. Ordered list   | ✅        |     |     |     |
| [link_example]()  | ✅        |     |     |     |
| Image (network)   | ✅        |     |     |     |
| - [x] Checkbox    | ❌        |     |     |     |
| Code              | ❌        |     |     |     |
| Blockquote        | ✅        |     |     |     |
| Font family       | ❌        |     |     |     |
| Text color        | ❌        |     |     |     |
